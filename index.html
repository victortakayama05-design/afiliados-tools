Como obter sua API Key

Para usar o CopyForge AI, você precisa de uma chave de API da OpenAI (modelo BYOK - Bring Your Own Key). Isso garante que você tenha controle total sobre seus custos e dados.

  1. Acesse platform.openai.com/api-keys.
  2. Faça login ou crie uma conta.
  3. Adicione créditos de saldo (Settings > Billing). Recomenda-se colocar $5 para começar.
  4. Clique no botão "Create new secret key".
  5. Copie a chave gerada (ela começa com sk-...) e cole no campo no topo desta página.

Sua chave fica salva apenas no seu navegador local (localStorage). Nós não temos acesso a ela e não há servidor intermediário. É 100% seguro.
